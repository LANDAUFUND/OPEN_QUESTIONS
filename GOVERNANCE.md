# GOVERNANCE

## Роль управления (governance)

Landau Fund строит открытую научную инфраструктуру (open scientific infrastructure), где GitHub является публичным журналом версий, обсуждений, вопросов, рецензий и решений.

## Human approval layer

AI-assisted research может ускорять анализ, но публичные научные статусы, утверждения, рейтинги, индексы, рецензии, выводы и публикации требуют человеческого утверждения (human approval).

## Решения

- ключевые решения фиксируются в версиях GitHub;
- спорные утверждения получают статус гипотезы или открытого вопроса;
- отрицательные результаты публикуются;
- ошибки архивируются;
- патентно-рискованные технические детали не раскрываются до отдельной проверки patent layer;
- пожертвования и финансирование должны быть прозрачны.

## GitHub features

Для каждого публичного репозитория должны быть включены Issues, Discussions, Pull Requests, Wiki и Projects. При отсутствии удаленного репозитория это требование фиксируется как настройка к выполнению после создания GitHub Organization.
