# CONTRIBUTING

## Как участвовать

Участие возможно через Issues, Discussions, Pull Requests, рецензии, экспериментальные протоколы, библиографические карточки, индексы, отрицательные результаты и открытые вопросы.

## Научная дисциплина

Не выдавайте гипотезу (hypothesis) за доказанный факт. Разделяйте закон (law), теорему (theorem), формулу (formula), модель (model), метод (method), гипотезу (hypothesis), эксперимент (experiment), аномалию (anomaly) и интерпретацию (interpretation).

## Pull Requests

Каждый Pull Request должен содержать:

- цель изменения;
- статус утверждений;
- источники данных;
- ограничения;
- указание AI-assisted participation, если AI участвовал;
- отсутствие секретов, токенов, cookies, private keys и закрытых патентно-рискованных деталей.
