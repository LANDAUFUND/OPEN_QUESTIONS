# REPOSITORY SETTINGS

После создания удаленного репозитория на GitHub необходимо включить:

- Issues;
- Discussions;
- Pull Requests;
- Wiki;
- Projects.

Рекомендуемые настройки:

- default branch: main;
- visibility: public;
- license: CC BY 4.0;
- citation: CITATION.cff;
- funding: через .github/FUNDING.yml, если применимо;
- branch protection: включить после первой стабилизации команды.
