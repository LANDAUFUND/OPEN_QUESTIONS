# PROJECT BOARD

GitHub Project должен иметь следующие колонки или статусные поля:

1. IDEAS
2. HYPOTHESES
3. FORMULAS
4. MODELS
5. SIMULATIONS
6. EXPERIMENTS
7. REVIEW
8. PUBLISHED
9. ARCHIVE

## Назначение

Project Board фиксирует путь идеи от первичной постановки до публикации, рецензии или архива. Перемещение карточки не является доказательством научной истинности; оно отражает только рабочий статус.
